Thanks for installing Sync Settings

With Sync Settings, you are able to synchronize your settings among multiple devices, and keep them updated.
Sync Settings, is powered by Github-Gist, providing you a secure and reliable way to store your backups.

Getting started

1. Go to https://github.com/settings/tokens/new and create a new token with `gist` scope
2. if Do you already have a gist?
     Copy gist id and put it in config file
     Run `Download` command
   else
     Run `Create and upload` command

Issues

If you are experimenting an error or an unusual behavior, please let me know, reporting the error creating a issue on https://github.com/mfuentesg/SyncSettings/issues/new.
Please, run `Show logs` command to complement the issue and find the bug faster.
